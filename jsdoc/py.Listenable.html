addListener(eventName, callback)

Function adds a listener callback to the Listenable object, to listen to the target custom event. This class should be used as a Mixin when creating object which can trigger custom events.
Parameters:
Name Type Description
eventName String
callback Function
Source:
  • Listenable.js, line 16
Example
py.Class('MyClass',{ Mixin: py.Listenable, 
   execute: function(){
       this.dispatchEvent('onExecuteCall');
   }
},'pckg');

var myInstance = new pckg.MyClass();
myInstance.addListener('onExecuteCall', function(){
   //handle custom event 
});
//...
myInstance.execute();
